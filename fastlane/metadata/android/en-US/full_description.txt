GuitarTuner is a free, open-source, offline acoustic guitar tuner for Android. It listens through the phone microphone, identifies the strummed string, and gives clear tune-up or tune-down feedback.

Features:
- Auto mode detects whichever open string you play
- Guided mode walks through each string step by step
- Chromatic mode shows the nearest note for any pitch
- Reference tone playback lets you tune by ear
- Custom tunings via JSON import/export
- Built-in tunings: Standard, Half-step down, Drop D, Open G, DADGAD
- A4 calibration (default 440 Hz), adjustable cents tolerance, noise gate
- Per-string peg-direction calibration for tune-up/tune-down guidance
- String-break overshoot warning when pitch is dangerously high
- Haptic in-tune confirmation (optional)
- Freeze last note display after note decay
- AMOLED dark theme with light and system options
- Live microphone diagnostics (RMS, peak, source, sample rate)
- TalkBack accessible tuning meter with live region announcements

Privacy:
- Microphone audio is processed locally only and never leaves the device
- No network permission, no ads, no accounts, no tracking
- Audio samples are not stored, exported, or uploaded
- Preferences stay in local app storage
