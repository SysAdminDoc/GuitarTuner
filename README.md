# GuitarTuner

![Version](https://img.shields.io/badge/version-v0.0.1-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![Platform](https://img.shields.io/badge/platform-Android%208.0%2B-lightgrey)

GuitarTuner is a planned open-source Android app for tuning an acoustic guitar by sound. The app will listen through the microphone, identify the strummed open string, show whether it is flat or sharp, and guide the player through small tuning adjustments.

## Planned Direction

- Native Android app with Kotlin, Jetpack Compose, and Material 3.
- Offline-first design with no network permission.
- Microphone-only pitch detection using Android audio capture.
- Guided standard-tuning workflow for acoustic guitar.
- Automatic string detection with confidence and noise feedback.
- Clear tune-up / tune-down guidance with optional peg-direction calibration.
- AMOLED dark theme by default with an accessible light theme.

## Repository Status

GuitarTuner v0.0.1 is a planning baseline. No Android project files have been generated yet. The next step is the scaffold pass in [ROADMAP.md](ROADMAP.md).
